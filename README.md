# Xservice — статический веб-проект

Xservice — это статический сайт сервиса с адаптивной версткой, SEO-оптимизацией  
и базовым интерактивом на JavaScript.

Проект не требует серверной части и может быть размещён на любом хостинге.

---

## 🚀 Возможности

- Адаптивная верстка (ПК / планшеты / смартфоны)
- Семантическая HTML5-разметка
- SEO-оптимизация (meta-теги, sitemap, robots.txt)
- Модальные окна и формы обратной связи
- Плавные анимации интерфейса
- Оптимизированная структура файлов

---

## 📁 Структура проекта

/
├── index.html # Главная страница
├── robots.txt # Правила для поисковых роботов
├── sitemap.xml # Карта сайта
├── assets/
│ ├── css/
│ │ ├── tokens.css # Дизайн-токены
│ │ └── main.css # Основные стили
│ ├── js/
│ │ └── main.js # Логика интерфейса
│ ├── img/ # Изображения
│ └── fonts/ # Локальные шрифты (опционально)
