# Xservice — статический проект

## Структура проекта

- `index.html` — главная страница (семантика, SEO meta, модалка и форма)
- `robots.txt` — правила для поисковых роботов
- `sitemap.xml` — карта сайта
- `assets/css/tokens.css` — дизайн-токены
- `assets/css/main.css` — стили компонентов
- `assets/js/main.js` — интерактив (меню, модалки, форма, анимации)
- `assets/img/` — изображения (можно добавить `og.jpg`/`favicon`)
- `assets/fonts/` — локальные шрифты (опционально)

## Быстрый старт (локально)

Это статический сайт. Достаточно открыть `index.html` в браузере.

Для корректной проверки относительных путей и поведения (например, модалки/scroll) лучше запускать локальный сервер:

- VS Code: расширение Live Server
- Любой другой простой static-server
